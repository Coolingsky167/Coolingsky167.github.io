排序算法总结

发表于2021-07-18|更新于2021-07-19|Leetcode
|阅读量:

插入排序

简单插入排序

  • 从第一个元素开始，该元素可以认为已经被排序
  • 取下一个元素（要比较的元素），在已经排序的元素序列中从后向前扫描
  • 如果扫描到的元素大于要比较的元素，则后移要比较的元素一位
  • 重复上一步，直到扫描到的元素小于等于要比较的元素
  • 将元素插入到该位置后
  • 重复2-5

希尔排序

与插入排序不同的是优先比较距离较远的元素，又叫缩小增量排序

交换排序

冒泡排序

  • 比较相邻的元素，如果第一个比第二个大，就交换
  • 针对每一对相邻元素计算，每次循环后最大的元素到末尾
  • 重复执行上述步骤

快速排序

通过一趟排序将待排记录分隔成独立的两部分，其中一部分记录的关键字均比另一部分的关键字小，则可分别对这两部分记录继续进行排序，以达到整个序列有序。

  • 用分治的思想把一串分成两个子串
  • 从数列中挑出一个元素，称为基准(pivot)
  • 重新排序数列，将所有比基准小的元素放置在基准前，元素比基准大的放在后面
  • 递归地操作

选择排序

简单选择排序

首先在未排序序列中找到最小(大)元素，放在排序序列的起始位置，然后再从剩余未排序的元素中，寻找最小(元素)，放在已排序序列的末尾，直到所有元素均排序完毕。

堆排序

堆(heap)

堆分为两种：最大堆和最小堆，两者的差别在于节点的排序方式。

在最大堆中，父节点的值比每一个子节点的值都要大。在最小堆中，父亲节点的值比每一个子节点的值都要小。这就是所谓的“堆属性”，并且这个属性对堆中的每一个节点都成立。

堆排序算法

大顶堆：arr[i] >= arr[2i+1] && arr[i] >= arr[2i+2]

小顶堆：arr[i] <= arr[2i+1] && arr[i] <= arr[2i+2]

建立大根堆

把所有非叶节点都检查一遍是否满足大根堆的要求，不满足就进行调整

归并排序

  • 把长度为n的输入序列分成两个长度为n/2的子序列
  • 对这两个子序列分别采用归并排序
  • 将两个排序好的子序列合成最终排序序列
